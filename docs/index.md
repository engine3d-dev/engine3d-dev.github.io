## Overview

Engine3D is an open-source game engine project. A project students can contribute to and have a tool they are familiar with to develop their own games.

🚧 NOTE: Documentation Website Still In-Progress 🚧
 

Purpose behind Engine3D:

1. Portability and Cross-platform support.
2. Multi-targeted
3. Safe, Reliable, and Testable or can be Tested.
4. Easy build and game application startup.

## Why did you create Engine3D?
Engine3D was at first a personal project developed by Aaron (SpinnerX) who at the time only wanted Engine3D to be a personal project. Firstly developed using OpenGL. As he continued developing the predecessors of Engine3D, it became too difficult to maintain. Realizing this, he decided to make this project open-source where students can contrribute and have a tool they can build as part of their learning experiences.
